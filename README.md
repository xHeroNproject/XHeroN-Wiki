# X Hero N Wiki

Неофициальная энциклопедия карты **X Hero N 10.9.2** для Warcraft III.

## Сайт

Статический сайт без серверной части. Поддерживает:

- русский и английский языки;
- Classic и Reforged;
- героев, способности, предметы, боссов, системы и секреты;
- патчноуты и загрузки актуальной карты 10.9.2.

## Публикация через GitHub Pages

1. Откройте **Settings → Pages** в репозитории.
2. В разделе **Build and deployment** выберите **Deploy from a branch**.
3. Выберите ветку **main** и папку **/(root)**.
4. Нажмите **Save**.

После публикации адрес будет иметь вид:

`https://USERNAME.github.io/REPOSITORY/`

## Обновление сайта

Замените изменённые файлы в локальной папке, создайте commit в GitHub Desktop и нажмите **Push origin**. GitHub Pages автоматически опубликует новую версию после обновления ветки `main`.
