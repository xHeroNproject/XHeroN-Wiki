X Hero N Wiki 1.1.6 — пакет обновления

Этот архив содержит только файлы, изменённые для публикации X Hero N 10.9.1.

Порядок установки:
1. В GitHub Desktop восстановите удалённые файлы репозитория: вкладка Changes → правый клик по списку → Discard all changes.
2. Закройте сайт/редактор, если файлы открыты.
3. Распакуйте СОДЕРЖИМОЕ этого архива прямо в корень репозитория XHeroN-Wiki, где находится папка .git.
4. Согласитесь на замену существующих файлов.
5. В GitHub Desktop должно появиться 12 изменений.
6. Создайте commit и нажмите Push origin.

Не удаляйте содержимое репозитория перед установкой патча.
